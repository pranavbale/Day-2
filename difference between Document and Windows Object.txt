What is document object ?
The document object is your html, aspx, php, or other document that will be loaded into the browser.
The document actually gets loaded inside the window object and has properties available to it like
title, URL, cookie, etc. What does this really mean? That means if you want to access a property 
for the window it is window.property, if it is document it is window.document.property which is also 
available in short as document.property.

what is Windows Object ?
The window object represents a window in browser. An object of window is created automatically by 
the browser. Window object is the object of browser.
